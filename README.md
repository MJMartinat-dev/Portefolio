## Compétences

### Géomatique & Données spatiales
- Administration **PostgreSQL / PostGIS** — modélisation, requêtes spatiales avancées, optimisation
- **QGIS · ArcGIS · GeoServer** — analyses, publications WMS/WFS, SLD
- Cartographies thématiques et interactives (**Leaflet**)
- Traitement de vastes jeux de données géographiques

### Développement applicatif
- **R · RShiny · Golem** — applications web modulaires et déployables
- **RMarkdown** — reporting automatisé multi-formats (HTML, PDF, LaTeX)
- **HTML · CSS · JavaScript** — habillage et interactions front-end
- **SQL** — requêtes complexes, pipelines de traitement

### DataViz & Graphisme
- Cartographies thématiques · posters · dataviz statique et interactive
- Photogrammétrie · modélisation 3D (SketchUp · MNT)
- Suite Adobe · Illustrator · Impression grand format

### Gestion de projet
- Conduite de A à Z (conception → déploiement → documentation)
- Coordination inter-institutionnelle (DDT, DREAL, IGN)
- Versioning GitLab · documentation technique orientée reprise externe

## Posture professionnelle

- **Documentation systématique** — chaque livrable est accompagné de sa documentation technique
- **Souci de maintenabilité** — code structuré pour être repris par d'autres agents ou équipes
- **Orientation utilisateur** — outils conçus pour des non-développeurs (agents DDT, partenaires)
- **Autonomie technique** — de l'analyse du besoin à la mise en production
